URLs

These documentation pages are no longer maintained. Please visit the new documentation site.

The following table contains a list of template operators related to URLs and location strings. Click on the name of an operator for more information about that specific operator (usage, parameters, description, examplets, etc.).

Operator Description
ezdesignPrefixes input string with current design directory.
ezimagePrefixes input string with image directory in current design.
ezrootSame as the "ezurl" operator but doesn't include "index.php".
ezurlMakes sure that a URL works in both VH and NVH environments.
exturlManages URLs using the URL database
