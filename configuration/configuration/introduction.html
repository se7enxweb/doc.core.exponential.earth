Configuration files

eZ publish can be configured with a set of configuration files. Configuration files are placed in the settings/ directory in the eZ publish root directory on the server. The main configuration file is called site.ini and is located directly in the settings/ directory. site.ini is divided up in several blocks. A block name is placed inside brackets.

Example:

[DatabaseSettings]

DatabasePluginPath=

DatabaseImplementation=ezmysql

Server=localhost

User=publishuser

Password=publishpass

Database=publishdb

UseSlaveServer=disabled

ConnectRetries=0

Charset=iso-8859-1

UseBuiltinEncoding=true

Socket=disabled

SQLOutput=enabled

UsePersistentConnection=disabled

 

[RoleSettings]

EnableCaching=true

PolicyOmitList[]=user

PolicyOmitList[]=layout

PolicyOmitList[]=manual

PolicyOmitList[]=ezinfo

In this example we have two blocks, DatabaseSettings and RoleSettings. Each block have several variables that decide the behaviour of our site. Some of the variables has a name ending with [], this means that the value is treated as an array, and we can add as menu values to this variable as we want.
