Change the right configuration file

These documentation pages are no longer maintained. Please visit the new documentation site.

As described in Configuration files and directories explained, you have to make any changes to your configuration in the correct ini files. /settings/override overide /settings/siteaccess which override the standard files /settings.

So usually, you would not want to change anything in the standard config files in /settings but rather in /settings/siteaccess (per logical site) and in /settings/override (for overall settings).
