About the documentation

These documentation pages are no longer maintained. Please visit the new documentation site.

This section contains information about the structure and status of the eZ publish documentation pages. It also sheds light on some issues regarding comments and contributions. Currently, the documentation only exists in the English language. A multilanguage structure will be probably put into place as soon as the English version reaches a mature state.

The eZ publish documentation is written by employees of eZ systems and by community members. This means that every registered user on ez.no has the ability to edit existing and to create new documentation pages. For more information about making changes to the documentation, please refer to the guidelines section. All documentation is available for free. Some chapters also exist in PDF version. The PDFs can be downloaded from the main documentation page. Please keep in mind that the PDF versions might not be as up-to-date as the online version of the documentation.
